# Icons

Drop the following PNGs into this directory before publishing to the Chrome Web Store:

- `icon-16.png` — 16×16 (toolbar)
- `icon-48.png` — 48×48 (extensions page)
- `icon-128.png` — 128×128 (Web Store)

Until they're added, Chrome uses a default puzzle-piece icon. Developer-mode installs work without icons present.

Brand source: `tako-core/frontend/public/logo-horizontal.svg` — extract the mark and export at each size.
